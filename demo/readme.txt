sshforwarding.ssh:      ssh command to execute in order to connect to the reasoner environment (usually ssh login into a virtual machine)
getversion.sh:          command to execute within the reasoner environment in order to retrieve the reasoner version
reasonercall.sh:        command to execute within the reasoner environment in order to call the reasoner (HEX-program will be sent via stdin, plugin via option --pythonplugin=)
