Experiments

This page describes experiments conducted mainly with dlvhex version 2.x. It contains information how to obtain and build dlvhex 2.x, as well as measurements for

  • comparing dlvhex 2.x with the previous version dlvhex 1.7.3,
  • comparing different parameter settings of dlvhex 2.x, and
  • comparing dlv and clingo backends of dlvhex 2.x.

dlvhex 2.x

dlvhex 2.x is not yet released, but you can get it from sourceforge using subversion:

  • you find the branch online here: dlvhex/branches/dlvhex-refactoring,
  • checking out and building the sources works as follows:

    $ svn co https://dlvhex.svn.sf.net/svnroot/dlvhex/dlvhex/branches/dlvhex-refactoring
$ cd dlvhex-refactoring
$ ./bootstrap.sh
<ignore all warnings>
$ ./configure <insert your parameters here>
$ make
$ make check
<optional>
$ make install
<optional>
$ sudo ldconfig
<required on some machines>
