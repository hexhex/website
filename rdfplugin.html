The RDF Plugin

The RDF Plugin provides a single atom. It takes a single constant as input, which denotes the RDF-source, and returns rdf-triples:

&rdf[I](X,Y,Z)
Input:
I
web address or file path
Output:
X,Y,Z
RDF triples

Example: 

triple(S,P,O) :- &rdf["http://example.org/foaf.rdf"](S,P,O).
